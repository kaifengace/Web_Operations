###  用户访谈

设定的问题：

- 用户偏好亮色网站背景？

访谈结果：有4位受访者表明更为喜欢偏明亮的网站风格。他们认为偏明亮的风格可以使网站显得更为简洁明了、干净整齐能够增加网站的可读性同时还能使网站显得比较正式正规。有2位受访者喜欢偏暗的风格，他们认为偏暗的色调，使网站显得很高档很酷，而且在晚上浏览网站时也没有那么伤眼睛。

解决方法：最终选择了偏明亮风格的网站模板

- 在目录页面，是喜欢文章只显示概述而不是直接显示全文，还是直接显示全文？

访谈结果：6位受访者都喜欢在目录页面，文章只显示概述。他们认为这样可以使页面更为简洁以及文章排序更为整齐，这样内容就更容易找到而且增加了可读性。

解决方法：在目录页面，是喜欢文章只显示概述

- 喜欢文章是纯文字的还是带有配图的

访谈结果：3位受访者喜欢文章是纯文字的，他们认为配图会分散他们的注意力，降低了他们的阅读质量以及阅读速度。有3位受访者喜欢文章内带有配图的，他们认为如果文章的字数较多，适当的插入几张养眼的图片可以缓解一下视觉疲劳，而且有的配图还能帮助用户理解文章的内容。

解决方法：当文章的字数较多或者文章内容较为难理解，则可适当加入配图帮助用户缓解视觉疲劳或者理解文章内容。如果文章内容不算多且内容较容易被理解的话，则可以不加入配图。

- 是否喜欢文章内附有外部链接（外部链接主要是文章内容的扩展）

访谈结果：大部分受访者表示无所谓，认为有无外部链接都可以。有1位受访者明确表示喜欢文章内附有外部链接，因为她认为这样可以使文章内容更具可靠性同时还能获得更多的知识。

解决方法：在文章内加入外部链接。

- 是否喜欢外部链接是嵌入文章内的（例如：点击文章内的图片可以跳转至外部网页或点击文章内的某个词可以跳转至外部页面），还是放置在文章的底部等明显的位置？

访谈结果：有3位受访者喜欢将外部链接放在文章底部等明显的位置，他们认为这样比较方便，不需要额外地花费时间精力四处寻找外部链接。有3位受访者认为直接将外部链接放置在文章底部等明显的位置有点影响页面的整洁性以及影响页面的美观度。

解决方法：为了方便用户的操作，还是选择将外部链接放置于文章底部。同时，我也有将外部链接或者网站的内部链接嵌入性地放置文章的内容里（文章内的配图或者字段中的某个词），主要也是为了方便用户浏览到更多的资讯，增强网站的可操作性。
